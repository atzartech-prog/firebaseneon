Setup Firebase

Hubungkan aplikasi dengan Firebase Project Anda untuk memulai

Tempel seluruh konfigurasi SDK Web dari console Firebase Anda.

Cara Mendapatkan Kunci:

  1. Buka dashboard Firebase Console.
  2. Buat proyek baru atau pilih proyek yang sudah ada.
  3. Klik ikon roda gigi ⚙️ > **Project settings**.
  4. Di bagian bawah, buat aplikasi Web baru (klik ikon ``).
  5. Salin objek JSON konfigurasi `firebaseConfig` dan tempel di atas!
  6. Aktifkan metode masuk **Email/Password** dan **Google** di menu **Authentication > Sign-in method** di console Firebase Anda agar login berfungsi.
